JavaScript — це мова програмування, яка додає веб-сторінкам логіку та інтерактивність.

Що це за мова

JavaScript дозволяє реагувати на кліки, змінювати HTML і CSS, працювати з даними, формами, таймерами та запитами.

Де використовується

У веб-сайтах, браузерних іграх, мобільних застосунках, десктопних програмах і на сервері через Node.js.

Як підключається

Код можна писати в тегу <script> або підключати окремий файл через src.

Що робить приклад

console.log() виводить повідомлення у консоль розробника браузера. Це зручно для перевірки коду.

<script>
  console.log("Привіт, світ!");
</script>

Коментарі

// створює однорядковий коментар, а /* ... */ — багаторядковий.

Регістр

JavaScript чутливий до регістру: name і Name — різні імена.

Крапка з комою

; завершує інструкцію. Часто працює і без неї, але для новачків краще ставити явно.

let, const, var

let — змінна, const — значення без повторного присвоєння, var — старий спосіб оголошення.

// Однорядковий коментар
/*
  Багаторядковий коментар
*/

let age = 18;
const userName = "Олена";
var oldStyle = true;

Тип даних описує, яке значення зберігається у змінній: число, текст, логічне значення, об’єкт або список.
