Показано всі теми довідника.

C — це компільована мова програмування, яка дає багато контролю над пам’яттю і роботою програми.

Де використовується

C використовують у системному програмуванні, embedded, драйверах, операційних системах, інструментах командного рядка та ігрових рушіях.

Чому C важлива

C допомагає зрозуміти, як працює пам’ять, адреси, масиви, вказівники, компіляція і низькорівневі частини комп’ютера.

Переваги

C швидка, компактна, близька до заліза, має простий синтаксис і величезну кількість існуючого коду.

Складність

У C потрібно уважно працювати з пам’яттю, вказівниками, рядками і межами масивів. Мова дає свободу, але вимагає дисципліни.

Перша програма

#include <stdio.h>

int main()
{
    printf("Привіт, світ!\n");
    return 0;
}
  • #include <stdio.h> підключає стандартну бібліотеку вводу і виводу.
  • int main() оголошує головну функцію програми.
  • { } містять тіло функції.
  • printf виводить текст у консоль.
  • \n додає перехід на новий рядок.
  • return 0 повідомляє, що програма завершилась успішно.

// створює коментар до кінця рядка, а /* */ — багаторядковий коментар.

Крапка з комою

Більшість інструкцій у C завершується символом ;.

int age = 18;
printf("%d", age);

Фігурні дужки

{ } об’єднують інструкції в блок: функцію, умову або цикл.

if (age >= 18) {
    printf("OK");
}

#include, main, return

#include підключає заголовковий файл, main() є точкою входу, return повертає результат.

#include <stdio.h>

int main() {
    return 0;
}

Змінні

Змінна має тип, назву і може мати початкове значення.

  • Оголошення: int x;
  • Ініціалізація: int x = 10;
  • const забороняє зміну значення.
  • sizeof повертає розмір типу або змінної в байтах.
const int max = 100;
printf("%zu", sizeof(max));

Оператори

Оператори виконують обчислення, порівняння, логіку і присвоєння.

  • Арифметичні: + - * / %
  • Порівняння: == != > < >= <=
  • Логічні: && || !
  • Присвоєння: = += -= *= /=
  • Інкремент і декремент: ++ --
int x = 5;
x += 2;
x++;

Умови

if, else if, else і switch дозволяють вибирати різний код.

if (score >= 90) {
    printf("Відмінно");
} else if (score >= 60) {
    printf("Зараховано");
} else {
    printf("Спробуй ще");
}

Цикли

Цикли повторюють блок коду.

  • for зручний для лічильника.
  • while працює, поки умова істинна.
  • do while виконується мінімум один раз.
  • break зупиняє цикл, continue переходить до наступної ітерації.
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}

Функції

Функція — це іменований блок коду з параметрами і результатом.

  • Прототип оголошує функцію до використання.
  • return повертає значення.
  • void означає, що результату немає.
int add(int a, int b);

int add(int a, int b) {
    return a + b;
}

Масиви

Масив зберігає кілька елементів одного типу під однією назвою.

  • Індекс починається з 0.
  • Одновимірний масив схожий на список.
  • Багатовимірний масив схожий на таблицю.
int numbers[3] = {1, 2, 3};
int matrix[2][2] = {{1, 2}, {3, 4}};

Рядки

Рядок у C — це масив char, який завершується символом '\0'.

  • printf виводить рядки.
  • scanf читає введення, але потребує обережності.
  • strlen рахує довжину.
  • strcpy копіює рядок.
  • strcmp порівнює рядки.
#include <string.h>

char name[20] = "Ivan";
printf("%zu", strlen(name));

Вказівники

Pointer зберігає адресу іншої змінної. Оператор & бере адресу, а * дістає значення за адресою.

int number = 10;
int *ptr = &number;

printf("%d", *ptr);

Масив і вказівник пов’язані: назва масиву часто поводиться як адреса першого елемента.

Структури

struct дозволяє створити власний тип з полями.

  • . відкриває поле звичайної структури.
  • -> відкриває поле через вказівник.
struct Player {
    char name[20];
    int health;
};

struct Player player = {"Alex", 100};
printf("%d", player.health);

Динамічна пам’ять

Динамічна пам’ять виділяється під час роботи програми.

  • malloc виділяє пам’ять.
  • calloc виділяє і заповнює нулями.
  • realloc змінює розмір.
  • free звільняє пам’ять.
  • Завжди звільняй пам’ять через free().
  • Не використовуй пам’ять після free().
int *items = malloc(5 * sizeof(int));
if (items != NULL) {
    items[0] = 10;
    free(items);
}

Файли

C працює з файлами через тип FILE.

  • fopen відкриває файл.
  • fclose закриває файл.
  • fprintf записує форматований текст.
  • fscanf читає форматовані дані.
  • fgets читає рядок.
  • fputs записує рядок.
FILE *file = fopen("data.txt", "w");
if (file != NULL) {
    fprintf(file, "Hello");
    fclose(file);
}

Препроцесор

Препроцесор обробляє директиви перед компіляцією.

  • #include підключає файли.
  • #define створює макрос або константу.
  • #ifdef перевіряє, чи визначено ім’я.
  • #ifndef перевіряє, чи ім’я не визначено.
  • #endif завершує умовний блок.
#ifndef CONFIG_H
#define CONFIG_H

#define MAX_SIZE 100

#endif
