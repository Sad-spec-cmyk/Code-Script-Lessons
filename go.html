Показано всі теми довідника.

Go, або Golang, це компільована мова програмування від Google, створена для простого, швидкого і надійного серверного коду.

Що це за мова

Go має простий синтаксис, швидку компіляцію, статичну типізацію і стандартні інструменти для форматування, тестування та збірки.

Де використовується

Go часто використовують для бекенду, серверів, мікросервісів, DevOps-інструментів, CLI-програм, API та мережевих сервісів.

Переваги

Мова цінується за швидкість, простоту, зрозумілий код, вбудовану concurrency-модель, goroutines і канали.

fmt.Println()

Функція з пакета fmt виводить текст у консоль і додає перехід на новий рядок.

package main

import "fmt"

func main() {
    fmt.Println("Привіт, світ!")
}
  • package main оголошує виконуваний пакет програми.
  • import "fmt" підключає пакет для форматованого вводу і виводу.
  • func main() є точкою входу, з якої починається програма.
  • fmt.Println друкує повідомлення у консоль.

package main

func

Змінні

var оголошує змінну явно, := коротко створює змінну всередині функції, const створює константу.

var name string = "Olena"
age := 21
const appName = "Shop"

Оператори

Go має арифметичні, логічні оператори та оператори порівняння.

  • Арифметичні: + - * / %
  • Порівняння: == != > < >= <=
  • Логічні: && || !
score := 15
isAdult := age >= 18
canEnter := isAdult && score > 10

Умови

if і else дозволяють виконувати різний код залежно від умови.

if age >= 18 {
    fmt.Println("Доступ дозволено")
} else {
    fmt.Println("Доступ заборонено")
}

Цикли

У Go є один основний цикл — for. Він може працювати як класичний цикл, while-цикл або перебір колекції.

for i := 0; i < 5; i++ {
    fmt.Println(i)
}

Функції

Функція може мати параметри, повертати результат і навіть повертати кілька значень.

func divide(a int, b int) (int, error) {
    if b == 0 {
        return 0, fmt.Errorf("ділення на нуль")
    }
    return a / b, nil
}

Масиви і слайси

Array має фіксовану довжину, а slice гнучкіший і частіше використовується у реальному Go-коді.

numbers := []int{1, 2, 3}
numbers = append(numbers, 4)
fmt.Println(numbers[0])

Карти map

map[key]value зберігає пари ключ-значення. За ключем можна швидко отримати значення.

ages := map[string]int{
    "Anna": 22,
    "Ivan": 25,
}
fmt.Println(ages["Ivan"])

Структури

struct описує власний тип з полями. Це основа для моделювання даних у Go.

type User struct {
    Name string
    Age  int
}

user := User{Name: "Anna", Age: 22}

Методи

Метод — це функція з receiver. Receiver показує, до якого типу належить метод.

func (u User) SayHello() {
    fmt.Println("Привіт,", u.Name)
}

Інтерфейси

interface описує поведінку через набір методів. Тип реалізує інтерфейс автоматично, якщо має потрібні методи.

type Speaker interface {
    Speak() string
}

Goroutines

go запускає функцію паралельно. Goroutine легша за звичайний системний потік.

go func() {
    fmt.Println("Працюю паралельно")
}()

Канали channels

chan дозволяє goroutines обмінюватися даними. Оператор <- відправляє або приймає значення.

messages := make(chan string)

go func() {
    messages <- "done"
}()

fmt.Println(<-messages)

Робота з файлами

Пакет os дозволяє читати й записувати файли. Старий пакет ioutil часто зустрічається у старих прикладах.

data, err := os.ReadFile("data.txt")
if err != nil {
    fmt.Println(err)
}
fmt.Println(string(data))

Обробка помилок

У Go помилки зазвичай повертаються як значення типу error. Перевірка if err != nil є типовим стилем.

file, err := os.Open("data.txt")
if err != nil {
    return err
}
defer file.Close()
