Показано всі теми довідника.

Swift — це сучасна компільована мова програмування від Apple для створення швидких, безпечних і зручних застосунків.

Що це за мова?

Swift має строгі типи, зрозумілий синтаксис, Optional для безпечної роботи з відсутніми значеннями та добре підходить для великих застосунків.

Для чого використовується?

Swift використовують для iOS, macOS, watchOS, tvOS, серверних програм, інструментів і навчання сучасному програмуванню.

Переваги Swift

Мова швидка, безпечна, читабельна, має type inference, зручну роботу з колекціями, протоколами та сучасний фреймворк SwiftUI.

print()

Функція print() виводить текст або значення в консоль.

import Foundation

print("Привіт, світ!")
  • import Foundation підключає базову бібліотеку Foundation.
  • print() виводить повідомлення у консоль.
  • "Привіт, світ!" — це рядок тексту.

Коментарі

Однорядковий коментар починається з //, багаторядковий пишеться через /* */.

let

Оголошує константу. Значення не можна змінити після присвоєння.

let appName = "Notes"

var

Оголошує змінну, значення якої можна змінювати.

var score = 0
score = 10

import

Підключає модуль або фреймворк, наприклад Foundation чи SwiftUI.

import SwiftUI

Змінні і константи

let створює константу, var створює змінну. Swift часто сам визначає тип через type inference.

let name = "Olena"
var score = 0
score = 5

Оператори

Swift має арифметичні, логічні оператори, оператори порівняння та присвоєння.

  • Арифметичні: + - * / %
  • Порівняння: == != > < >= <=
  • Логічні: && || !
let canEnter = age >= 18 && score > 10

Умови

if, else і switch виконують різний код залежно від значення.

if age >= 18 {
    print("Доступ дозволено")
} else {
    print("Доступ заборонено")
}

Цикли

for-in перебирає колекції, while працює поки умова істинна, repeat-while виконується хоча б один раз.

for number in [1, 2, 3] {
    print(number)
}

Функції

func оголошує функцію. Функція може мати параметри й повертати результат через return.

func add(_ a: Int, _ b: Int) -> Int {
    return a + b
}

Optional

Optional означає, що значення може бути відсутнім. ? створює optional, if let безпечно дістає значення, ! примусово розгортає.

var name: String? = "Ivan"

if let userName = name {
    print(userName)
}

Масиви

Array зберігає список. append додає елемент, remove видаляє.

var items = ["Book", "Phone"]
items.append("Watch")
items.remove(at: 0)

Словники

Dictionary зберігає ключі та значення. Доступ до значення робиться через ключ.

var user = ["name": "Ivan", "city": "Kyiv"]
print(user["name"] ?? "Немає імені")

Структури і класи

struct і class описують власні типи. init налаштовує об’єкт під час створення.

struct Player {
    var name: String

    init(name: String) {
        self.name = name
    }
}

ООП

Інкапсуляція приховує деталі, наслідування дозволяє класу отримати поведінку іншого класу, поліморфізм дозволяє працювати через спільний тип.

class Animal {
    func speak() {
        print("Sound")
    }
}

class Dog: Animal {
    override func speak() {
        print("Woof")
    }
}

Protocol

protocol описує вимоги до типу. Структура або клас реалізує ці вимоги.

protocol Drivable {
    func drive()
}

struct Car: Drivable {
    func drive() {
        print("Їду")
    }
}

Обробка помилок

do, try і catch використовують для коду, який може кинути помилку.

do {
    try riskyAction()
} catch {
    print("Помилка")
}

SwiftUI

SwiftUI описує інтерфейс декларативно. Text показує текст, VStack складає елементи вертикально, Button створює кнопку.

struct ContentView: View {
    var body: some View {
        Text("Привіт")
    }
}

Калькулятор

func calculate(_ a: Double, _ b: Double, _ op: String) -> Double? {
    switch op {
    case "+": return a + b
    case "-": return a - b
    case "*": return a * b
    case "/": return b == 0 ? nil : a / b
    default: return nil
    }
}

print(calculate(10, 5, "+") ?? 0)
