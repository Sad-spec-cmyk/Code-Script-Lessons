Java — об’єктно-орієнтована мова програмування для Android, бекенду, великих корпоративних систем, сервісів і кросплатформних застосунків.

Що це за мова

Java є строго типізованою мовою. Код організовується у класи, а програма зазвичай починається з методу main.

Де використовується

Android, Spring Boot бекенд, фінансові системи, мікросервіси, великі enterprise-проєкти, десктопні й серверні застосунки.

JVM коротко

JVM, Java Virtual Machine, запускає скомпільований bytecode. Завдяки цьому Java може працювати на різних платформах.

System.out.println()

Виводить текст у консоль і переходить на новий рядок.

public class Main {
    public static void main(String[] args) {
        System.out.println("Привіт, світ!");
    }
}
