C++ — потужна компільована мова для high-performance задач, системного програмування, ігор, рушіїв, графіки та програм, де важливий контроль над пам’яттю і швидкість.

Що це за мова

C++ поєднує процедурне програмування, ООП, шаблони і низькорівневий контроль. Вона швидка, але вимагає уважності до пам’яті й типів.

Де використовується

Ігри, ігрові рушії, системні утиліти, драйвери, фінансові системи, embedded, графічні програми, high-performance сервери.

#include <iostream>

Підключає бібліотеку для введення і виведення, зокрема cout.

using namespace std;

Дозволяє писати cout замість std::cout. У великих проєктах часто краще писати std:: явно.

int main()

Головна функція програми. З неї починається виконання.

return 0;

Повертає код успішного завершення програми операційній системі.

#include <iostream>
using namespace std;

int main()
{
    cout << "Привіт, світ!" << endl;
    return 0;
}
