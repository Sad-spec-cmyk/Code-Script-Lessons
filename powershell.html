Показано всі теми довідника.

PowerShell — це командна оболонка і скриптова мова для автоматизації задач, особливо у Windows та адмініструванні систем.

Де використовується

PowerShell використовують для роботи з файлами, процесами, службами, реєстром, серверами, Azure, Windows-адмініструванням і автоматизацією повторюваних задач.

CMD і PowerShell

CMD переважно працює з текстом і старими командами. PowerShell працює з об’єктами, має pipeline, функції, модулі і зручні cmdlets у стилі Verb-Noun.

Перший приклад

Write-Output "Привіт, світ!"
  • Write-Output виводить значення у pipeline.
  • "Привіт, світ!" — рядок тексту.
  • Команду можна запускати у PowerShell-консолі або в .ps1 скрипті.

Команди cmdlets

Cmdlet — це стандартна команда PowerShell. Багато команд повертають об’єкти, а не просто текст.

Get-Process
Get-Service

Verb-Noun стиль

Назви команд часто мають формат дієслово-іменник: Get-Process, Set-Location, New-Item.

Get-Command -Verb Get

Коментарі

# створює коментар до кінця рядка. Багаторядковий коментар пишеться через <# #>.

# Однорядковий коментар
<#
Багаторядковий коментар
#>

Pipeline

Символ | передає результат однієї команди в іншу.

Get-Process | Sort-Object CPU -Descending

Приклад
Get-Help Get-Process

Опис
Шукає доступні команди.
Приклад
Get-Command *Service*

Опис
Показує запущені процеси.
Приклад
Get-Process

Опис
Показує служби Windows.
Приклад
Get-Service

Опис
Показує файли і папки. Схоже на ls або dir.
Приклад
Get-ChildItem

Опис
Переходить в іншу папку. Схоже на cd.
Приклад

Змінні і типи

Змінні починаються зі знака $. PowerShell може визначати тип автоматично, але тип можна вказати явно.

$name = "Іван"
$age = 25
[int]$count = 10

Оператори

PowerShell має арифметичні, логічні та оператори порівняння.

  • Арифметичні: + - * / %
  • Порівняння: -eq -ne -gt -lt -ge -le
  • Логічні: -and -or -not
$age -ge 18
$isAdmin -and $isActive

Умови

if і else виконують різний код залежно від умови.

if ($age -ge 18) {
    "Доступ дозволено"
} else {
    "Доступ заборонено"
}

Цикли

for, foreach і while повторюють команди.

foreach ($file in Get-ChildItem) {
    $file.Name
}

Функції

function створює власну команду. Параметри можна описувати через param.

function Get-Greeting {
    param($Name)
    return "Привіт, $Name"
}

Get-Greeting -Name "Оля"

Масиви і об’єкти

@() створює масив, а [PSCustomObject] створює простий об’єкт з властивостями.

$items = @("one", "two")

$user = [PSCustomObject]@{
    Name = "Іван"
    Age = 25
}

Робота з файлами

Get-Content читає файл, Set-Content записує новий вміст, Add-Content додає текст у кінець.

Set-Content -Path "log.txt" -Value "Start"
Add-Content -Path "log.txt" -Value "Done"
Get-Content -Path "log.txt"

Pipeline

Pipeline передає об’єкти між командами. Це дозволяє фільтрувати, сортувати і форматувати дані.

Get-Process |
    Where-Object CPU -gt 10 |
    Sort-Object CPU -Descending

Скрипти .ps1

PowerShell-скрипти зберігаються у файлах .ps1. Запускати локальний скрипт можна через .\script.ps1.

.\backup.ps1

Обробка помилок

try містить ризиковану дію, а catch реагує на помилку.

try {
    Get-Content "missing.txt" -ErrorAction Stop
} catch {
    "Файл не знайдено"
}

Список процесів

Get-Process |
    Select-Object Name, Id, CPU |
    Sort-Object CPU -Descending

Робота з файлами

New-Item -Path "notes.txt" -ItemType File
Set-Content -Path "notes.txt" -Value "Перша нотатка"
Add-Content -Path "notes.txt" -Value "Друга нотатка"
Get-Content -Path "notes.txt"

Автоматизація задачі

$source = "C:\Logs"
$target = "C:\Archive"

Get-ChildItem $source -Filter "*.log" |
    Copy-Item -Destination $target

Скрипт

param(
    [string]$Name = "Гість"
)

Write-Output "Привіт, $Name"

Назва: Get-Help

Опис
Показує довідку по команді.
Синтаксис
Get-Help CommandName
Приклад
Get-Help Get-Service
Примітка
Додавайте -Examples для прикладів.

Назва: Get-Command

Опис
Шукає команди, функції та aliases.
Синтаксис
Get-Command pattern
Приклад
Get-Command Get-*
Примітка
Корисно, коли не пам’ятаєте назву команди.

Назва: Get-Process

Опис
Повертає процеси.
Синтаксис
Get-Process [-Name name]
Приклад
Get-Process powershell
Примітка
Можна передавати в pipeline.

Назва: Get-Service

Опис
Повертає служби.
Синтаксис
Get-Service [-Name name]
Приклад
Get-Service Win*
Примітка
Для керування службами є Start-Service і Stop-Service.

Назва: Get-ChildItem

Опис
Показує вміст папки.
Синтаксис
Get-ChildItem path
Приклад
Get-ChildItem -Recurse
Примітка
Alias: ls, dir, gci.

Назва: Set-Location

Опис
Змінює поточну папку.
Синтаксис
Set-Location path
Приклад
Set-Location C:\Temp
Примітка
Alias: cd.

Назва: Where-Object

Опис
Фільтрує об’єкти у pipeline.
Синтаксис
Where-Object condition
Приклад
Where-Object CPU -gt 10
Примітка
Alias: where, ?.

Назва: ForEach-Object

Опис
Виконує дію для кожного об’єкта в pipeline.
Синтаксис
ForEach-Object { script }
Приклад
Get-ChildItem | ForEach-Object { $_.Name }
Примітка
$_ означає поточний об’єкт.

Назва: Get-Content

Опис
Читає вміст файлу.
Синтаксис
Get-Content path
Приклад
Get-Content log.txt
Примітка
Може читати файл построково.

Назва: Set-Content

Опис
Записує новий вміст у файл.
Синтаксис
Set-Content -Path path -Value value
Приклад
Set-Content notes.txt "Hello"
Примітка
Перезаписує файл.

Назва: Add-Content

Опис
Додає текст у кінець файлу.
Синтаксис
Add-Content -Path path -Value value
Приклад
Add-Content log.txt "Done"
Примітка
Не стирає попередній вміст.
