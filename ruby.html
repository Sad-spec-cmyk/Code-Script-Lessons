Показано всі теми довідника.

Ruby — це динамічна об’єктно-орієнтована мова програмування з простим синтаксисом, яка часто читається майже як звичайний текст.

Що це за мова?

Ruby створена для зручного написання коду. У Ruby майже все є об’єктом: числа, рядки, масиви, класи й навіть помилки.

Де використовується?

Ruby використовують у веб-розробці, автоматизації, скриптах, тестуванні, CLI-інструментах і особливо у фреймворку Ruby on Rails.

Особливості Ruby

Мова цінується за читабельність, гнучкість, блоки, ітератори, просту роботу з колекціями та сильний фокус на зручності розробника.

puts

Метод puts виводить текст у консоль і додає перехід на новий рядок.

puts "Привіт, світ!"
  • puts друкує значення у консоль.
  • "Привіт, світ!" — це рядок тексту.
  • У Ruby не потрібно писати крапку з комою після кожної інструкції.

puts

Виводить текст або значення. Часто використовується у перших прикладах і для простого налагодження.

puts "Hello"

Коментарі

Однорядковий коментар починається з символу #. Коментарі пояснюють код і не виконуються.

# Це коментар
puts "Ruby"

end

Закриває блоки коду: методи, класи, умови, цикли та інші конструкції.

if true
  puts "OK"
end

Змінні

Локальні змінні починаються з маленької літери або підкреслення. Тип визначається автоматично.

name = "Olena"
age = 21

Назва: Integer

Опис
Ціле число без дробової частини.
Приклад
age = 25
Коли використовувати
Для кількості, віку, індексів і лічильників.

Назва: Float

Опис
Число з дробовою частиною.
Приклад
price = 19.99
Коли використовувати
Для вимірювань, цін і приблизних розрахунків.

Назва: String

Опис
Текстовий рядок.
Приклад
name = "Ivan"
Коли використовувати
Для імен, повідомлень, адрес і будь-якого тексту.

Назва: Boolean

Опис
Логічне значення true або false.
Приклад
active = true
Коли використовувати
Для умов, прапорців і перевірок.

Назва: Array

Опис
Впорядкована колекція значень.
Приклад
numbers = [1, 2, 3]
Коли використовувати
Коли потрібно зберігати список елементів.

Назва: Hash

Опис
Колекція пар ключ-значення.
Приклад
user = { name: "Ivan" }
Коли використовувати
Для структурованих даних з іменованими полями.

Назва: Symbol

Опис
Легке незмінне ім’я, яке часто використовують як ключ.
Приклад
:name
Коли використовувати
Для ключів у hash, назв станів і службових ідентифікаторів.

Змінні

Локальні змінні зберігають дані всередині методу або блоку. Константи пишуть з великої літери, глобальні змінні починаються з $.

name = "Ivan"
PI = 3.14
$debug = true

Оператори

Ruby має арифметичні, логічні оператори та оператори порівняння.

  • Арифметичні: + - * / % **
  • Порівняння: == != > < >= <=
  • Логічні: && || !
score = 15
adult = age >= 18
can_enter = adult && score > 10

Умови

if, elsif, else та case виконують різний код залежно від умови.

if age >= 18
  puts "Доступ дозволено"
elsif age >= 16
  puts "Потрібен дозвіл"
else
  puts "Доступ заборонено"
end

Цикли

while, until, for та each повторюють код. У Ruby часто використовують each.

[1, 2, 3].each do |number|
  puts number
end

Методи

Метод оголошується через def, може мати параметри й повертати значення. Останній вираз часто повертається автоматично.

def add(a, b)
  a + b
end

puts add(2, 3)

Масиви

Array зберігає список. push додає елемент, pop забирає останній, each перебирає.

items = ["book", "pen"]
items.push("phone")
items.pop
items.each { |item| puts item }

Hash

Hash зберігає ключі та значення. Доступ до значення робиться через ключ.

user = { name: "Ivan", age: 25 }
puts user[:name]
user[:age] = 26

Рядки

Інтерполяція вставляє значення у рядок через #{}. Рядки мають методи upcase, downcase, strip, split.

name = "Ruby"
puts "Hello, #{name}"
puts name.upcase

ООП у Ruby

Клас описує об’єкт. initialize запускається під час створення, attr_accessor створює читання й запис властивості.

class Player
  attr_accessor :name

  def initialize(name)
    @name = name
  end
end

player = Player.new("Olia")

Модулі

module групує методи або константи. include додає методи модуля до класу.

module Greeting
  def hello
    puts "Hello"
  end
end

class User
  include Greeting
end

Робота з файлами

File.open, read і write дозволяють читати й записувати файли.

File.open("data.txt", "w") do |file|
  file.write("Hello")
end

text = File.read("data.txt")

Обробка помилок

begin, rescue та ensure дозволяють обробляти помилки й виконувати завершальний код.

begin
  number = Integer("abc")
rescue ArgumentError
  puts "Невірне число"
ensure
  puts "Готово"
end

Ruby on Rails

Rails — популярний веб-фреймворк на Ruby. Він використовує MVC: Model для даних, View для інтерфейсу, Controller для логіки запитів.

class ArticlesController
  def index
    @articles = Article.all
  end
end

Калькулятор

def calculate(a, b, operator)
  case operator
  when "+"
    a + b
  when "-"
    a - b
  when "*"
    a * b
  when "/"
    b == 0 ? "Помилка" : a / b
  else
    "Невідомий оператор"
  end
end

puts calculate(10, 5, "+")

Опис
Оголошує метод.
Синтаксис

Синтаксис
