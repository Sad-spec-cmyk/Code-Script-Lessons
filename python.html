Розділи

Вибір мови Що таке Python Синтаксис Типи даних Теми Практичні приклади Довідник Пошук

Python — популярна мова програмування з простим синтаксисом, яку часто обирають для навчання, автоматизації, вебу, аналізу даних і скриптів.

Що це за мова

Python читається майже як звичайний текст. Він динамічно типізований: тип значення визначається під час виконання.

Де використовується

Скрипти, веб-застосунки, автоматизація, ігри, тестування, боти, аналіз даних, машинне навчання і системні утиліти.

Переваги

Простий синтаксис, велика стандартна бібліотека, багато пакетів, швидкий старт і сильна спільнота.

print()

Функція print() виводить текст або значення в консоль. Рядок пишеться в лапках.

print("Привіт, світ!")

Відступи

Python використовує відступи для позначення блоків коду. Зазвичай використовують 4 пробіли.

Коментарі

# створює коментар до кінця рядка. Коментарі не виконуються програмою.

Регістр

Python чутливий до регістру: name і Name — різні імена.

Блоки коду

Після if, for, while, def, class ставиться двокрапка.

# Це коментар
name = "Олена"

if name == "Олена":
    print("Привіт!")

У Python змінну не потрібно оголошувати з типом. Тип визначається за значенням.
