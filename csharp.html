Розділи

Вибір мови Основи C# Синтаксис Типи даних Теми Практичні приклади Довідник Пошук

C# — сучасна об’єктно-орієнтована мова програмування від Microsoft. Її часто використовують для .NET-застосунків, бекенду, десктопних програм, Unity-ігор і сервісів.

Що таке C#

Це строго типізована мова: змінні, методи й об’єкти мають визначені типи. Такий підхід допомагає знаходити багато помилок ще до запуску програми.

Де використовується

C# застосовують у Unity, ASP.NET Core, .NET MAUI, Windows-програмах, API, іграх, мікросервісах і корпоративних системах.

using System;

Підключає простір імен System, де знаходиться клас Console.

class Program

Оголошує клас. У C# майже весь код знаходиться всередині класів або інших типів.

static void Main()

Точка входу в консольну програму. З неї починається виконання коду.

Console.WriteLine()

Виводить текст у консоль і переходить на новий рядок.

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Привіт, світ!");
    }
}

Коментарі

// створює однорядковий коментар, а /* ... */ — багаторядковий.

Регістр

C# чутливий до регістру: name і Name — різні імена.

Структура коду

Код зазвичай організований у простори імен, класи, методи й блоки у фігурних дужках.

using і namespace

using підключає простір імен, а namespace групує ваші класи.

// Однорядковий коментар
/*
   Багаторядковий коментар
*/

using System;

namespace MyApp
{
    class Program
    {
        static void Main()
        {
            Console.WriteLine("C#");
        }
    }
}

Тип даних визначає, які значення може зберігати змінна і які операції з ними можна виконувати.
