Показано всі теми довідника.

Rust — це сучасна системна мова програмування, яка поєднує швидкість C/C++ з суворими правилами безпеки пам’яті.

Де використовується

Rust застосовують у системному програмуванні, бекенді, CLI-інструментах, WebAssembly, high-performance сервісах та проєктах, де важлива надійність.

Переваги

Головні переваги Rust: безпека пам’яті без garbage collector, висока швидкість, сильна типізація, зручний менеджер пакетів Cargo.

Перша програма

fn main() {
    println!("Привіт, світ!");
}
  • fn main() оголошує головну функцію, з якої починається програма.
  • { } містять тіло функції.
  • println! — макрос, який виводить текст у консоль.

let і mut

let створює змінну. За замовчуванням вона незмінна. mut дозволяє змінювати значення.

let name = "Rust";
let mut score = 0;
score = 10;

Крапка з комою

Більшість інструкцій завершується символом ;. Вираз без ; може повертати значення.

let x = 5;
let y = {
    let a = 2;
    a + 3
};

Функції

fn оголошує функцію. Типи параметрів і результату зазвичай вказуються явно.

fn add(a: i32, b: i32) -> i32 {
    a + b
}

Rust є строго типізованою мовою. Компілятор часто може сам вивести тип, але у важливих місцях тип краще писати явно.
