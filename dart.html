Dart — це сучасна мова програмування від Google. Її часто використовують разом із Flutter для створення мобільних, веб і desktop застосунків з однієї кодової бази.

Що це за мова?

Dart має зрозумілий синтаксис, статичну типізацію, null safety і добре підходить для UI-застосунків.

Для чого використовується?

Мобільна розробка, Flutter, веб-застосунки, desktop-програми, серверні утиліти та прототипи.

Flutter

Flutter — це фреймворк для інтерфейсів. Dart описує логіку і віджети, а Flutter малює застосунок на різних платформах.

Переваги Dart

Швидкий старт, читабельний код, hot reload у Flutter, async/await, null safety і зручні колекції.

void main() {
  print("Привіт, світ!");
}
  • void main() — головна функція, з якої починається програма.
  • print() — виводить текст у консоль.
  • Крапка з комою завершує інструкцію.

// створює однорядковий коментар, /* */ — багаторядковий.

// Коментар

Точка входу в консольну Dart-програму.

print()

Виводить значення у консоль. Часто використовується для перевірки коду.

print("Hello");

Імпорт бібліотек

import підключає стандартні бібліотеки або пакети.

import 'dart:math';

Назва: int

Опис: ціле число.

Приклад: int age = 18;

Коли використовувати: лічильники, вік, кількість.

Назва: double

Опис: число з дробовою частиною.

Приклад: double price = 19.99;

Коли використовувати: ціни, координати, вимірювання.

Назва: String

Опис: текстовий рядок.

Приклад: String name = "Ivan";

Коли використовувати: імена, повідомлення, назви.

Назва: bool

Опис: логічне значення true або false.

Приклад: bool isReady = true;

Коли використовувати: перевірки, прапорці, стани.

Назва: List

Опис: впорядкований список значень.

Приклад: List<int> nums = [1, 2, 3];

Коли використовувати: масиви даних, списки товарів, елементи UI.

Назва: Map

Опис: колекція ключ-значення.

Приклад: Map<String, int> scores = {"Olia": 10};

Коли використовувати: налаштування, словники, дані користувача.

Назва: Set

Опис: колекція унікальних значень.

Приклад: Set<String> tags = {"dart", "flutter"};

Коли використовувати: унікальні теги, ролі, фільтри.

Назва: dynamic

Опис: змінна без фіксованого типу.

Приклад: dynamic value = 10;

Коли використовувати: рідко, коли тип справді невідомий.

Змінні

var визначає тип автоматично, final задає значення один раз, const створює compile-time константу, dynamic дозволяє змінювати тип.

var name = "Olena";
final createdAt = DateTime.now();
const appName = "Shop";
dynamic value = 10;

Оператори

Dart має арифметичні, логічні оператори, оператори порівняння та присвоєння.

  • Арифметичні: + - * / %
  • Порівняння: == != > < >= <=
  • Логічні: && || !
var total = 10 + 5;
var canEnter = age >= 18 && score > 10;

Умови

if, else, switch і ternary оператор виконують різний код залежно від умови.

if (age >= 18) {
  print("Доступ дозволено");
} else {
  print("Доступ заборонено");
}

var label = score > 10 ? "Добре" : "Спробуй ще";

Цикли

for, while, do while і for-in повторюють код.

for (var item in ["Dart", "Flutter"]) {
  print(item);
}

var count = 0;
while (count < 3) {
  count++;
}

Функції

Функція має назву, параметри й може повертати результат. Arrow syntax => підходить для коротких функцій.

int add(int a, int b) {
  return a + b;
}

int square(int n) => n * n;

Колекції

List зберігає порядок, Map працює з ключами, Set зберігає унікальні значення.

var names = ["Anna", "Ivan"];
var user = {"name": "Olia", "city": "Kyiv"};
var tags = {"dart", "flutter"};

Null Safety

? дозволяє null, ! примусово розгортає значення, non-nullable типи не можуть бути null.

String? name = "Ivan";

if (name != null) {
  print(name);
}

Класи

class описує тип, constructor створює object, methods описують поведінку.

class Player {
  String name;

  Player(this.name);

  void sayHello() {
    print("Привіт, $name");
  }
}

ООП

Інкапсуляція приховує деталі, наслідування дозволяє розширити клас, поліморфізм дає працювати через спільний тип.

class Animal {
  void speak() => print("...");
}

class Dog extends Animal {
  @override
  void speak() => print("Woof");
}

Async

Future представляє майбутній результат, async дозволяє використовувати await.

Future<String> loadName() async {
  await Future.delayed(Duration(seconds: 1));
  return "Dart";
}

Обробка помилок

try, catch і finally допомагають обробляти помилки без падіння програми.

try {
  var number = int.parse("abc");
  print(number);
} catch (error) {
  print("Помилка");
} finally {
  print("Готово");
}

void main() {

Назва: var

Опис: змінна з автоматичним визначенням типу.

Синтаксис: var name = value;

Приклад: var age = 18;

Примітка: після визначення тип зазвичай не змінюється.

Назва: final

Опис: значення можна присвоїти один раз.

Синтаксис: final name = value;

Приклад: final id = DateTime.now();

Примітка: добре для значень, які відомі під час виконання.

Назва: const

Опис: compile-time константа.

Синтаксис: const name = value;

Приклад: const pi = 3.14;

Примітка: значення має бути відоме заздалегідь.

Назва: List

Опис: список елементів.

Синтаксис: List<Type> items = [];

Приклад: List<String> names = ["Olia"];

Примітка: індекси починаються з 0.

Назва: Map

Опис: структура ключ-значення.

Синтаксис: Map<K, V> data = {};

Приклад: {"name": "Ivan"}

Примітка: зручно для JSON-подібних даних.

Назва: Future

Опис: результат, який з’явиться пізніше.

Синтаксис: Future<Type>

Приклад: Future<String> load()

Примітка: часто використовується з async/await.

Назва: class

Опис: опис власного типу.

Синтаксис: class Name {}

Приклад: class User {}

Примітка: основа ООП у Dart.

Назва: Widget

Опис: базовий елемент інтерфейсу Flutter.

Синтаксис: Widget build(...)

Приклад: Text("Hello")

Примітка: майже все у Flutter є віджетом.
