Показано всі теми довідника.

PHP — це серверна мова програмування, яку часто використовують для створення динамічних веб-сайтів і бекенду.

Де використовується

PHP працює на сервері, обробляє запити, форми, сесії, файли, бази даних і генерує HTML для браузера.

Як працює з HTML

PHP-код можна вставляти у HTML-файл. Сервер виконує PHP, а користувач отримує готовий HTML.

Перший приклад

<?php
echo "Привіт, світ!";
?>
  • <?php відкриває PHP-код.
  • echo виводить текст.
  • ?> закриває PHP-блок, якщо після нього йде HTML.

PHP-теги

PHP-код пишеться всередині <?php ?>. Якщо файл містить тільки PHP, закривальний тег часто не пишуть.

<?php
echo "Hello";

Коментарі

PHP підтримує //, # і /* */.

// Коментар
# Ще один коментар
/* Багаторядковий коментар */

Крапка з комою

Більшість інструкцій завершується символом ;.

$name = "Іван";
echo $name;

echo і print

echo і print виводять текст. echo частіше використовується для простого виводу.

echo "Привіт";
print "Світ";

Змінні у PHP починаються зі знака $. Тип часто визначається автоматично за значенням.

Назва: string

Опис
Рядок тексту.
Приклад
$name = "Оля";
Коли використовувати
Для імен, повідомлень, HTML і текстових значень.

Назва: int

Опис
Ціле число.
Приклад
$age = 25;
Коли використовувати
Для кількості, id, віку і лічильників.

Назва: float

Опис
Дробове число.
Приклад
$price = 19.99;
Коли використовувати
Для цін, вимірювань і дробових розрахунків.

Назва: bool

Опис
Логічне значення true або false.
Приклад
$isAdmin = false;
Коли використовувати
Для умов, станів і прапорців.

Назва: array

Опис
Масив значень.
Приклад
$items = ["a", "b"];
Коли використовувати
Для списків, наборів даних і асоціативних структур.

Назва: object

Опис
Екземпляр класу.
Приклад
$user = new User();
Коли використовувати
Для ООП, моделей, сервісів і складної логіки.

Назва: null

Опис
Відсутність значення.
Приклад
$user = null;
Коли використовувати
Коли значення ще немає або воно спеціально очищене.

Оператори

PHP має арифметичні, порівняння, логічні та оператори присвоєння.

  • Арифметичні: + - * / %
  • Порівняння: == === != !== > <
  • Логічні: && || !
  • Присвоєння: = += -= *= /=
$total = 10 + 5;
$isAdult = $age >= 18;

Умови

if, else, elseif і switch дозволяють виконувати різний код.

if ($age >= 18) {
    echo "Доступ дозволено";
} elseif ($age >= 16) {
    echo "Майже";
} else {
    echo "Доступ заборонено";
}

Цикли

Цикли повторюють код. foreach особливо зручний для масивів.

$names = ["Анна", "Іван"];

foreach ($names as $name) {
    echo $name;
}

Функції

Функція — це іменований блок коду з параметрами і результатом.

function add($a, $b) {
    return $a + $b;
}

echo add(2, 3);

Масиви

Indexed array має числові індекси, associative array має ключі-рядки.

$colors = ["red", "green"];
$user = [
    "name" => "Іван",
    "age" => 25
];

Робота з формами

$_GET читає дані з URL, а $_POST — дані, відправлені формою методом POST.

$name = $_POST["name"] ?? "";
echo htmlspecialchars($name);

Суперглобальні змінні

Суперглобальні доступні майже всюди в PHP-скрипті.

  • $_SERVER містить інформацію про сервер і запит.
  • $_SESSION зберігає дані користувача між запитами.
  • $_COOKIE читає cookie браузера.
session_start();
$_SESSION["user_id"] = 10;

Робота з файлами

PHP може відкривати, читати, записувати і закривати файли.

$file = fopen("data.txt", "w");
fwrite($file, "Hello");
fclose($file);

База даних через PDO

PDO — універсальний спосіб працювати з базами даних. Для MySQL потрібні DSN, логін і пароль.

$pdo = new PDO(
    "mysql:host=localhost;dbname=test;charset=utf8",
    "root",
    ""
);

$stmt = $pdo->query("SELECT * FROM users");

SELECT і INSERT

SELECT читає дані, INSERT додає новий запис. Для даних користувача краще використовувати prepared statements.

$stmt = $pdo->prepare("INSERT INTO users (name) VALUES (?)");
$stmt->execute([$name]);

Обробка помилок

try містить ризикований код, а catch обробляє помилку.

try {
    $pdo = new PDO($dsn, $user, $pass);
} catch (PDOException $error) {
    echo "Помилка підключення";
}

Назва: echo

Опис
Виводить текст або значення.
Синтаксис
echo value;
Приклад
echo "Hello";
Примітка
Може виводити кілька значень через кому.

Назва: print

Опис
Виводить одне значення.
Синтаксис
print value;
Приклад
print "Hello";
Примітка
Повертає 1, але для простого виводу частіше використовують echo.

Назва: function

Опис
Оголошує функцію.
Синтаксис
function name($param) { }
Приклад
function hi() { echo "Hi"; }
Примітка
Функція може повертати значення через return.

Назва: foreach

Опис
Перебирає масив.
Синтаксис
foreach ($items as $item) { }
Приклад
foreach ($users as $user) echo $user;
Примітка
Можна отримувати ключ і значення.

Назва: $_GET

Опис
Масив параметрів з URL.
Синтаксис
$_GET["name"]
Приклад
$page = $_GET["page"] ?? 1;
Примітка
Не довіряйте даним користувача без перевірки.

Назва: $_POST

Опис
Масив даних POST-форми.
Синтаксис
$_POST["field"]
Приклад
$email = $_POST["email"] ?? "";
Примітка
Вивід треба екранувати через htmlspecialchars.

Назва: fopen

Опис
Відкриває файл.
Синтаксис
fopen(path, mode)
Приклад
fopen("data.txt", "r")
Примітка
Може повернути false.

Назва: fwrite

Опис
Записує дані у файл.
Синтаксис
fwrite($file, $text)
Приклад
fwrite($file, "Hello");
Примітка
Файл має бути відкритий у режимі запису.

Назва: PDO

Опис
Клас для роботи з базами даних.
Синтаксис
new PDO($dsn, $user, $pass)
Приклад
$pdo = new PDO($dsn, "root", "");
Примітка
Використовуйте prepared statements для даних користувача.

Назва: try / catch

Опис
Обробляє винятки.
Синтаксис
try { } catch (Exception $e) { }
Приклад
catch (PDOException $e) { echo "Error"; }
Примітка
Корисно для БД, файлів і ризикованих операцій.
